🐙 Manual: Crear una cuenta en GitHub

Guía de Inicio Rápido

A continuación, te presentamos los pasos detallados para crear tu cuenta en GitHub de forma rápida y sencilla:

  1. Ingresa a https://github.com/.
  2. Haz clic en el botón **Sign up** (Registrarse) para iniciar el proceso.
  3. Proporciona tu **correo electrónico**, crea un **nombre de usuario** y una **contraseña segura**.
  4. Completa la verificación (captcha) para confirmar que no eres un bot 🤖.
  5. Selecciona el plan gratuito (**Free**) a menos que prefieras explorar las opciones de pago.
  6. Confirma tu correo electrónico haciendo clic en el enlace de verificación que recibirás en tu bandeja de entrada.
  7. ¡Listo! 🎉 Ahora tienes una cuenta en GitHub.
